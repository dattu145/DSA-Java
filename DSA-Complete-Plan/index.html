All DSA Patterns - Complete Reference

Master these patterns to solve any DSA problem efficiently. ★ indicates most important patterns.

1. Two Pointers ★★

  • Pair sum (Two Sum sorted)
  • Remove duplicates
  • Palindrome check
  • Reverse array/string

2. Sliding Window ★★

  • Longest substring without repeating characters
  • Max sum subarray of size k
  • Minimum window substring
  • Fixed & Variable size windows

3. Fast & Slow Pointers ★

  • Cycle detection (Linked List)
  • Middle of linked list
  • Detect palindrome in linked list
  • Tortoise-Hare algorithm

4. Binary Search ★

  • Search in sorted array
  • Search on answer (min/max threshold)
  • Peak element
  • Rotated sorted array

5. HashMap / HashSet ★

  • Two sum
  • Anagrams
  • Frequency counting
  • Detect duplicates

6. Prefix Sum & Prefix XOR

  • Subarray sum queries
  • Count subarrays with certain sum/XOR
  • Range queries
  • Difference arrays

7. Monotonic Stack

  • Next greater element
  • Largest rectangle in histogram
  • Daily temperatures
  • Stock span problem

8. Priority Queue / Heap ★

  • Top K elements
  • Merge k sorted lists
  • K closest points
  • Dijkstra's Algorithm

9. Backtracking ★

  • Subsets
  • Permutations
  • N-Queens
  • Combination sum

10. Dynamic Programming ★★

  • 1D DP (Fibonacci, Climbing stairs)
  • 2D DP (LCS, Edit distance)
  • Knapsack patterns
  • DP on grids

11. Tree Traversals ★

  • DFS (Pre, In, Post order)
  • BFS / Level Order
  • Height / Depth
  • Path sum problems

12. Graph Traversals ★

  • DFS / BFS
  • Connected components
  • Flood fill
  • Cycle detection

13. Topological Sort

  • Course schedule
  • Dependency resolution
  • Detect cycle in directed graph

14. Union-Find (Disjoint Set)

  • Cycle detection
  • Connected components
  • Kruskal's MST

15. Trie (Prefix Tree)

  • Autocomplete
  • Word search
  • Prefix matching

16. Bit Manipulation

  • Finding single number using XOR
  • Checking power of two
  • Subset generation using bitmasks

17. Greedy Algorithms

  • Interval scheduling
  • Activity selection
  • Huffman coding

18. Advanced Graph Algorithms

  • Dijkstra's Algorithm
  • Bellman-Ford
  • Floyd-Warshall
  • Minimum Spanning Tree

19. String Matching Algorithms

  • KMP Algorithm
  • Rabin-Karp
  • Z-Algorithm

20. Advanced Data Structures

  • Segment Trees
  • Fenwick Trees
  • Suffix Arrays
  • LRU Cache
